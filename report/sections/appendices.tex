\section{On Simulation Stability}

Maintaining constant quantities as carefully as possible is necessary for 
successfully reproducing real-world conditions in simulation.
Damping parameters must be chosen to balance between rapid changes which
destabilise the system and slow changes which fail to sufficiently stabilise the 
conditions.
